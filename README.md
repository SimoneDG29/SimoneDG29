# Featured Projects

## Production-Grade DevSecOps Pipeline for Cloud-Native Microservices
🔗 GitHub: https://github.com/SimoneDG29/Secure-DevSecOps-Pipeline

A production-style DevSecOps pipeline for a polyglot microservices architecture, focused on CI/CD automation, software supply chain security, and Kubernetes deployment validation.

### Key Impact
- End-to-end CI/CD pipeline using GitHub Actions for a 5-service microservices system
- Full software supply chain security (Trivy, SBOM generation, Cosign signing, immutable image digests)
- Policy-as-code enforcement using OPA/Rego before build execution
- In-cluster admission control with Kyverno to enforce image signature verification and deployment security policies
- Shift-left security integration (SAST, DAST, dependency scanning, secret detection)
- CI/CD performance optimization with dynamic matrix generation and caching strategies

### Core Skills
DevSecOps engineering · Kubernetes security · CI/CD design · Policy as Code (OPA + Kyverno) · Software supply chain security

## GitHub Secret History Scanner (Go CLI Tool)
🔗 GitHub: https://github.com/SimoneDG29/Github-Secret-History-Scanner

A Go-based CLI tool that performs full Git history scanning to detect leaked secrets using regex + entropy-based detection.

Unlike standard scanners, this tool analyzes the **entire commit history**, making it suitable for security auditing and CI/CD enforcement.

### Key Impact
- Full Git history scanning (not only working tree)
- Custom YAML-based detection rules engine
- CI/CD ready (GitHub Actions integration)
- High-performance concurrent scanning
- Structured JSON output for automation pipelines

### Core Skills
Go · Secure SDLC · Static analysis · Security automation · DevSecOps tooling
